# 🎓 Student Scoreboard App

A React-based web application to manage student scores with dynamic updates, built using **React (Vite)** and **pure CSS**.

---

## 🚀 Features

- 📋 View list of students and their scores
- ➕ Add new students
- ✏️ Update student scores manually using button
- ❌ Delete students
- ✅ Pass / Fail status (based on score)
- 📊 Statistics Dashboard:
  - Total Students
  - Passed Students
  - Average Score
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (JSX)
- CSS (Pure CSS)

---

## 🧩 Components Used

- Header
- StudentTable
- StudentRow (Reusable)
- AddStudentForm
- Stats

---

## ⚙️ Functionality

- State management using `useState`
- Props used for data flow
- Conditional rendering for pass/fail
- Dynamic UI updates without reload

---

## 📊 Pass Criteria

- Pass → Score ≥ 40  
- Fail → Score < 40
